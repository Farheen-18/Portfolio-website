Carbon footprint forecasting using Quantum RNN
Technologies Used: Python, Qiskit, TensorFlow, NumPy, Pandas
Project Description:
Developed a Quantum Recurrent Neural Network (RNN) to forecast carbon footprints using quantum computing techniques.
The project involved data preprocessing, model training, and evaluation of the quantum RNN's performance against classical models.
The results demonstrated the potential of quantum computing in enhancing predictive accuracy for environmental data.

Vehicle Parking System with Verilog Gatel level Verilog Modelling
Technologies Used: Verilog, Quartus
Project Description:
Designed a vehicle parking system using Verilog at the gate level, simulating the parking logic and control mechanisms.
The project included the development of a user interface for parking management and real-time monitoring of parking slots.
The system was tested for efficiency and reliability in managing parking spaces in a simulated environment.
